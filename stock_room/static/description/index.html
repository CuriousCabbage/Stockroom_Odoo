Stockroom

Simplified Inventory System for Cafeterias with Multiple Outlets

This module is free for now as part of the pre-release. It will become paid once fully tested and stable.

Key Highlights

Centralized Stock Control

Manage shared storerooms and freezers across multiple outlets.

Expiry Tracking

Easily track expiry dates and get notified before items go bad (FIFO logic).

Usage & Insights

Monitor daily stock usage and generate insights on wastage or demand trends.

Simple & Intuitive UI

Designed for non-technical cafe staff with a clean layout and minimal learning curve.
